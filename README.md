# Интернет-магазин одежды XWEAR

### Стек технологий 
Бэкенд: Python, Django, DRF, PostgreSQL
Фронтенд (в процессе разработки): React 19, Vite, TanStack Query, Axios, SASS/SCSS, PostCSS.

### Реализовано на бэкенде:
- Регистрация пользователей по email и паролю с подтверждением почты. Реализована возможность сброса забытого пароля с использованием зарегистрированной почты пользователя.
- Авторизация/аутентификация пользователей с помощью JWT-токенов
- Кастомная админка с различными фильтрами, поиском, сортировкой, превью изображений, настройкой баннеров для слайдера главной страницы, возможностью создания многоуровневого меню сайта (любой степенью вложенности) и другим функционалом и визуализацией.
- Автоматическое создание миниатюр загруженных изображений товара, их переименование по названию товара и конвертирование в формат WebP. При удалении изображения или самого товара - соответствующие им миниатюры также автоматически удаляются.
- Фильтрация товаров по различным параметрам сайдбара сайта.
- Кабинет пользователя.
- Добавление товаров в избранное.
- Показ рекомендаций.
- Оптимизация SQL-запросов.
- REST API 
  - Пользователи: 
      получение jwt-токенов при входе в аккаунт, обновление access-токена, регистрация пользователя, активация пользователя при подтверждении email, обновление пароля, сброс пароля, выход из аккаунта, кабинет пользователя, создание и получение списка адресов пользователя, редактирование и удаление адреса пользователя.
  - Магазин:
      получение списка баннеров слайдера для главной страницы, получение дерева категорий меню, получение списка товаров категории, получение деталей товара, рекомендации товаров, получение избранных товаров пользователя, добавление/удаление товара из избранного.
  - Заказы:
      получение корзины текущего пользователя, добавление товара в корзину или увеличение количества, изменение количества товара в корзине, удаление позиции из корзины, оформление заказа из корзины, получение списка всех заказов текущего пользователя, получение списка доступных ПВЗ в текущем городе.
  - Общие данные:
      получение списка городов для доставки, получение списка юр.документов, получение списка контактов, получение условий доставки и оплаты, получение информации о компании.
